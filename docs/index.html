xerces-wasm

v1.0.0

XML + XSD validation for JavaScript

Powered by Apache Xerces-C compiled to WebAssembly. 0 Runtime Dependencies.


npm install xerces-wasm

Quick Start

Works with in-memory strings or file paths.

import { validate, validateFiles } from "xerces-wasm";

// Validate strings
const result = await validate(xmlText, xsdText);

// Validate files (Node.js)
const fileResult = await validateFiles("./doc.xml", "./schema.xsd");

API Output

Validation separates basic XML syntax errors from complex Schema logic errors.

interface ValidationResult {
  valid: boolean;
  parseErrors: Diagnostic[];
  schemaErrors: Diagnostic[];
}
